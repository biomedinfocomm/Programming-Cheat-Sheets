#### Cell
    <html>
    $A$2: fixed column and row
    A$2: fixed row
    $A2: fixed column
    </html>

##### Custom number format
    <html>
    0: display zeros
    (use #.00 > 9.20)
    </html>

#### Omit extra zeros displayed 
    <html>
    (use #.## > 3.4)
    </html>

#### Keyboard short cuts
    <html>
    ctrl A > select entire table 
    ctrl Z > undo
    ctrl; > fixed date 
    ctrl shift; > fixed time
    = today() > current date
    = now() > current date and time
    </html>
